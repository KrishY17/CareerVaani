Identify your strong domains

You are required to take a aptitude test based on which the career likeliness in a particular subject is predicted.

  • Strengths are identified based on positive attributes in user responses.
  • Weaknesses are pinpointed where user exhibits limitations or gaps.
  • Opportunities are extracted from external factors indicated by the user.
  • Threats are assessed based on potential challenges mentioned.
  • Each element (Strengths, Weaknesses, Opportunities, Threats) is given weightage.

Advanced ML Algorithms

User provides data via questionnaire.Various models are used to get precise recommendations for the user

  • Utilizes Logistic Regression, Random Forest ,Gradient Boosting K-Nearest Neighbors, Neural Networks, Naive Bayes, Decision Trees, SVM Linear, SVM Polynomial for predictions
  • Each algorithm independently predicts career options.
  • Combines predictions through a voting mechanism.
  • Presents user with recommended career among specified options based on aggregated insights.

Get the sentiments right!

  • Categorizes sentiments into positive, negative, and neutral categories.

  • Provides deeper understanding of challenges and stressors faced by students.
  • Enables personalized guidance and interventions for individual student needs.
